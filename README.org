* Dotfiles
 This repo is a place for all my dotfiles.

** Vim
Files:
- ~.vimrc~
- ~plug.vim~ in ~/.vim/autoload/

Configs:
- Plugin manager : Vim-Plug
- ColorScheme    : koehler
- FileNavigation : NERDTree
- For other configs check ~.vimrc~

** Bash
Files:
- ~.bashrc~
- ~.bash_aliases~

Configs:
- ~.bashrc~ contains
  - Some shell and history options
  - source to ~.bash_aliases~ file
